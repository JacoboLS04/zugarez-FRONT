## 1. INTRODUCCIÓN

Bienvenido a Zugarez, una aplicación web diseñada para la gestión comercial y de inventario. Este manual proporciona instrucciones detalladas sobre cómo utilizar todas las funcionalidades disponibles en el sistema.

## 2. REQUISITOS DEL SISTEMA

- Navegador web actualizado (Google Chrome, Mozilla Firefox, Microsoft Edge, Safari)
- Conexión a internet
- Resolución de pantalla recomendada: 1366x768 o superior

Para desarrolladores:
- Node.js (versión 14.x o superior)
- npm (versión 6.x o superior)
- Espacio en disco: mínimo 500MB para la instalación completa

## 5. TIPOS DE USUARIO

El sistema cuenta con dos tipos de usuario:

1. **Cliente**: Acceso al catálogo de productos, carrito de compras y gestión de pedidos personales.

2. **Administrador**: Acceso completo al sistema, incluyendo gestión de inventario, usuarios, pedidos y configuración.

## 6. PROCESO DE AUTENTICACIÓN

1. En la página de inicio, seleccione "Iniciar Sesión" o "Registrarse" según corresponda.

2. Para iniciar sesión:
- Introduzca su nombre de usuario o correo electrónico
- Introduzca su contraseña
- Seleccione su tipo de usuario (Cliente o Administrador)
- Haga clic en "Entrar"

3. Para registrarse:
- Haga clic en "Registrarse"
- Complete el formulario con su nombre de usuario, correo electrónico y contraseña
- Seleccione su tipo de usuario (Cliente o Administrador)
- Haga clic en "Registrarse"

4. Verificación de código:
- En algunos casos, recibirá un código de verificación por correo electrónico
- Introduzca el código recibido para completar la autenticación

## 9. MÓDULOS ESPECÍFICOS

### 9.1 Gestión de Inventario

El módulo de inventario permite a los administradores:

1. **Productos**: Añadir, editar, eliminar y visualizar productos.
2. **Lotes**: Gestionar lotes de productos con fechas de caducidad.
3. **Movimientos**: Registrar entradas y salidas de mercancía.
4. **Reportes**: Generar informes de inventario.
5. **Remisiones**: Gestionar documentos de remisión (ver sección 9.2).

Para acceder:
1. Inicie sesión como administrador
2. Navegue a la sección "Inventario" en el menú lateral
3. Seleccione el submódulo deseado

### 9.2 Remisiones de Inventario

Este módulo permite gestionar documentos de remisión en formato Excel:

1. **Visualizar remisiones**: 
- Al acceder al módulo, verá una lista de remisiones existentes
- Si no hay remisiones, verá un mensaje indicando que no hay documentos disponibles

2. **Añadir una remisión**:
- Haga clic en el botón "Añadir Remisión"
- Seleccione un archivo Excel (.xls o .xlsx) de su computadora
- El sistema validará que sea un formato Excel válido
- Una vez cargado, aparecerá en la lista de remisiones

3. **Gestionar remisiones existentes**:
- Descarga: Haga clic en el icono de descarga para obtener el archivo
- Eliminación: Haga clic en el icono de eliminar y confirme la acción

4. **Información mostrada**:
- Nombre del archivo
- Fecha de carga
- Tamaño del archivo
- Opciones de acción (descargar/eliminar)

## 10. SOLUCIÓN DE PROBLEMAS

### Problemas de autenticación
- Si no puede iniciar sesión con su usuario, intente con su correo electrónico.
- Si no recibe el código de verificación, revise su carpeta de spam.
- Si sigue sin recibir el código, contacte al soporte técnico.

### Errores al cargar remisiones
- Verifique que el archivo sea un Excel válido (.xls o .xlsx).
- El tamaño máximo permitido por archivo es de 10MB.

### Sesión cerrada inesperadamente
- El sistema cierra sesión automáticamente después de 60 minutos de inactividad por seguridad.
